# Telecom-Customer-Churn-Analysis-and-Prediction
Data analytics project showcasing customer churn analysis and Preidiction using SQL, Power BI , Python ans Machine Learning. This project showcases a complete ETL (Extract, Transform, Load) data pipeline  & a Power BI dashboard to utilize the Customer Data and achieve below goals:\
Visualize & Analyse Customer Data at below levels:Demographic,Geographic,Payment & Account Info, Services.\
Study Churner Profile & Identify Areas for Implementing Marketing Campaigns.\
Identify a Method to Predict Future Churners.\
STEP 1 – ETL Process in SQL Server(SQL): Used Microsoft SQL server to load the data , Executed Exploratory Data Analysis (EDA), checked for NUll values and replaced null values in a column, created views which are then consumed by POWER BI.\
STEP 2 – Visualization & Insights (Power BI): Connect Power BI to SQL server , create new DAX Measures and build an interactive dashboard that highlights key patterns and trends, enabling stakeholders to make data-driven decisions.\
STEP-3 - Predict Customer Churn(ML):For predicting customer churn,  I have used widely used Machine Learning algorithm called RANDOM FOREST where I performed Data Preprocessing( encoding), Model Creation and evaulating the model using ML Metrics. Then Model is used for Prediction on New Data and Visualized the predicted data using Power BI.
